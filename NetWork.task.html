NetWork.task¶

When a task is submited to the workgroup, you get a TaskHandler object that can be used to control the running task and receive information about its state.

class NetWork.task.TaskHandler(id, workgroup, worker)[source]¶

Class used to controll a running task and get information about it. A new instance is returned by Workgroup.submit method.

exception()[source]¶

Get the exception that the task has raised.

Return :exception that the task has raised, None if there was no exception.
exceptionRaised()[source]¶

Check if the task has raised an exception.

Return :True or False depending on whether the task has raised an exception.
result()[source]¶

Get return value of the submited function that’s running in this task.

Return :return value of the function in the task, None if the task hasn’t returned.
running()[source]¶

Check if the task is still running.

Return :True or False depending on whether the task is running.
terminate()[source]¶

Stop this task, kill its process.
